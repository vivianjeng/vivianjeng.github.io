Unity Games

  • Roll a Ball

    Roll a Ball Game Link
      learn more...

    The game is based on Unity Official Tutorial.

    but I made it more interesting.



    The ground is modified. We can't let the ball fall down the ground.

    There are yellow and red balls, if we touch the yellow one, we get one point,

    otherwise, we get "You Lose!" words.



    Falling down the ground will cause "You Lose!" words, too.

    If we get ten yellow balls, we get "You Win!" words.


    Skills:
    • Use arrow key to control.
    • Use OnTriggerEnter() to determine lose or not.
    • Use "transform.position.y" to determine falling down or not.

    How to Play?

    Use Arrow Key to control the white ball.

  • Maple Story

    Maple Story Game Link
      learn more...

    The game is based on Teacher's Tutorial.

    I selected Maple Story's characters and scenes to build this game.

    Because I love this kind of mission based RPG games, I created this game,

    and made the character solve the mission.

    Mission:
    1. Triggering the NPC.
    2. Collecting 30 mushroom caps from the monster.
    3. Giving the caps to the NPC.
    Skills:
    • A pickup to speed up.
    • A pickup to scale up.
    • A pickup to gain halo.
    • A trap to restart the game.
    • Instantiate monsters.
    • If we trigger the NPC at different moment, the NPC will say different words.

    How to play?

    Use Arrow Key "Right" and "Left" to move right and left.

    "Space" to jump

    "E" to pick up items

    "R" to restart the game

  • Kart Racing

    Game Download Link(for Windows)
      learn more...

    This is my "Basic Game Programming" final project.

    The game is inspired by KartRider(跑跑卡丁車),

    and the objects are from Unity Asset Store-Stylized Simple Cartoon City

    Rules:

    Two players compete racing kart.The first one who finishes the track, he win the game.

    There are a lot of gimmicks on the track, and the effects appear in random.

    Skills:
    • Items to speed up for 3 seconds.
    • Items to jump for 3 seconds.
    • Items to stop for 3 seconds.
    • Items to attack other player for 3 seconds.
    • Items to rotate for 3 seconds.
    • Items to mask oneself for 3 seconds.
    • Nav mesh agent to walk around.
    • 2D animations creating.
    • Adding a main menu.

    How to play?

    Use Arrow Key "Right" and "Left" to move right and left.

    Arrow Key "Up" to move forward, "Down" to move backward.

    "R" to restart the game
